PROCESS

1. Research

We ran a usability study to see what the biggest pain-points were in the user experience

I used the live site to conduct usability studies with 5 community members and used the results to make a priotized plan for improving the user experience. We immediately made a few changes to the existing site, then started preparting for a design overhaul.

2. Identifying the user funnel

I conducted a team session to map out the touchpoints and interactions with users

To better understand all the ways One Degree interacted with different audiences, I created a user experience map of how users were introduced to the organization, solved a problem or got value from the interaction, and then repeated this for a long-term relationship.
